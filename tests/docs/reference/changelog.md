# Changelog

This project uses [conventional commits](https://www.conventionalcommits.org/) and [python-semantic-release](https://python-semantic-release.readthedocs.io/) for automated versioning.

See [CHANGELOG.md](https://github.com/ralph089/mkdocs-modern-theme/blob/main/CHANGELOG.md) for the full release history.

## 0.1.0

Initial release.

- Dark mode with system preference detection
- Full-text search with Ctrl+K modal
- Responsive 3-column layout
- Geist font family (Sans + Mono)
- Admonition styling with 9 types (note, tip, warning, danger, info, abstract, success, bug, faq)
- Syntax highlighting for light and dark mode
- Collapsible sidebar navigation
- Table of contents with scroll-spy
- Breadcrumb navigation
- Previous/next page navigation
- CSS custom properties for theming
- Print styles and reduced motion support
- Content components: Steps, Cards, File Tree
- Mermaid diagram support with dark mode
- External link indicator on outbound links
- SVG logo and favicon support
