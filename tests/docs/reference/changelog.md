# Changelog

## 0.1.0 (Unreleased)

### Added

- Initial release
- Dark mode with system preference detection
- Full-text search with Ctrl+K modal
- Responsive 3-column layout
- Geist font family (Sans + Mono)
- Admonition styling (note, tip, warning, danger, info) with subtle shadows
- Syntax highlighting (light + dark) with shadow on code blocks
- Collapsible sidebar navigation
- Table of contents with scroll-spy
- Breadcrumb navigation
- Previous/next page navigation
- CSS custom properties for theming
- Print styles
- Reduced motion support
- Fumadocs-forward visual system: softer radii, elevation shadows, raised surface tokens
- Content components: Steps (numbered timeline), Cards (link grid with hover), File Tree (with folder/file icons)
- External link indicator icon on outbound links
- SVG logo and favicon support via `theme.logo` config
- Wider search bar with right-aligned keyboard shortcut hint
- Search result URLs resolve correctly from any page depth
- Self-documenting docs site with pages for all content components
