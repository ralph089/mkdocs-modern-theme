# Writing Documentation

This guide covers the basics of writing content with the Modern theme.

## Headings

Use standard Markdown headings. The theme automatically generates a table of contents from `h2` and `h3` headings.

## Text Formatting

You can use **bold**, *italic*, ~~strikethrough~~, and `inline code`.

> Blockquotes are styled with a left border accent.
