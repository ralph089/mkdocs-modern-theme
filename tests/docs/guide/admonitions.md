# Admonitions

Admonitions are visually distinct callout blocks for important content.

## Types

### Note

!!! note

    This is a note admonition. Use it for general information.

### Tip

!!! tip

    This is a tip. Use it for helpful suggestions.

### Warning

!!! warning

    This is a warning. Use it for potential issues.

### Danger

!!! danger

    This is a danger notice. Use it for critical warnings.

### Info

!!! info

    This is informational content.

!!! note "Custom Title Here"

    Admonitions support custom titles.
