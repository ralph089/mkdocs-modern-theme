# Typography

Modern ships with the [Geist](https://vercel.com/font) font family -- the same typeface used by Vercel and Next.js documentation. Both Geist Sans and Geist Mono are included as variable fonts, self-hosted from the theme package so no external requests are made.

## Default font stack

```css
:root {
  --modern-font-sans: "Geist", ui-sans-serif, system-ui, -apple-system, sans-serif;
  --modern-font-mono: "Geist Mono", ui-monospace, SFMono-Regular, "SF Mono", Menlo, monospace;
}
```

The sans stack is used for all body text, headings, and UI elements. The mono stack is used for inline code, fenced code blocks, and keyboard shortcuts.

## Switching to a different font

Override the font variables in your custom CSS. For example, to use Inter:

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap');

:root {
  --modern-font-sans: "Inter", ui-sans-serif, system-ui, -apple-system, sans-serif;
}
```

Or to use a system font stack without any web fonts:

```css
:root {
  --modern-font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
    "Helvetica Neue", Arial, sans-serif;
  --modern-font-mono: ui-monospace, SFMono-Regular, "SF Mono", Menlo,
    Consolas, "Liberation Mono", monospace;
}
```

!!! note "Geist still loads"

    The bundled Geist font files are declared with `@font-face` in the theme CSS. If you override `--modern-font-sans`, the browser will not download the Geist files since no element references them -- but the `@font-face` declarations remain. This has no performance impact.

## Switching the mono font

To use JetBrains Mono for code:

```css
@import url('https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500;600;700&display=swap');

:root {
  --modern-font-mono: "JetBrains Mono", ui-monospace, SFMono-Regular, monospace;
}
```

## Type scale

The theme uses a consistent type scale across content:

| Element     | Size              | Weight | Extra                                  |
| ----------- | ----------------- | ------ | -------------------------------------- |
| `h1`        | `2.25rem` (36px)  | 700    | Letter-spacing: -0.025em               |
| `h2`        | `1.5rem` (24px)   | 600    | Letter-spacing: -0.02em, bottom border |
| `h3`        | `1.25rem` (20px)  | 600    | --                                     |
| `h4`        | `1.125rem` (18px) | 600    | --                                     |
| `h5`, `h6`  | `1rem` (16px)     | 600    | --                                     |
| Body text   | `1rem` (16px)     | 400    | Line-height: 1.75                      |
| Inline code | `0.875em`         | 400    | Relative to parent                     |
| Code blocks | `0.875rem` (14px) | 400    | Line-height: 1.6                       |
| Table text  | `0.875rem` (14px) | 400    | --                                     |

## Heading styles

Headings are tight-tracked with negative letter-spacing at the larger sizes. `h2` elements include a subtle bottom border to create visual sections. All heading levels render in `--modern-text-primary`.

Permalink anchors (the `#` links generated by the `toc` extension) are hidden by default and fade in on hover:

```css
.modern-content .headerlink {
  opacity: 0;
  transition: opacity 150ms;
}
.modern-content h2:hover .headerlink {
  opacity: 1;
}
```

## Line height and spacing

Body text uses a line-height of `1.75` for comfortable reading at screen distance. Paragraphs have `1.25rem` bottom margin. Lists have the same bottom margin with `0.25rem` spacing between items.

## Font rendering

The theme enables subpixel anti-aliasing on macOS and grayscale anti-aliasing on other platforms:

```css
body {
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

This produces sharper, thinner text that matches the aesthetic of modern documentation sites.

## About Geist

Geist is a variable font, meaning a single file covers weights from 100 to 900. The theme primarily uses weights 400 (body), 500 (tab labels, UI), 600 (headings, bold content), and 700 (h1). The variable font files are approximately 57 KB (sans) and 58 KB (mono) in woff2 format -- smaller than loading four separate static weight files.
